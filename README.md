## Run Locally

Clone the project

```bash
  git clone https://github.com/DevEssaAhmed/Pxie.git
```

Go to the project directory

```bash
  cd CRUD-Recipe-App-React
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm  run dev
```
